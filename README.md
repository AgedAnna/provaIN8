## Instalação

Para rodar o projeto, é necessário baixar suas dependências. No terminal, execute o seguinte comando:

```bash
npm install
```

## Executando o Projeto

No terminal, execute o seguinte comando:

Isso iniciará o projeto. Abra http://localhost:3000 no seu navegador para visualizá-lo.

```bash
npm  start
```

Este README fornece instruções claras e diretas sobre como instalar e executar o projeto. Certifique-se de adaptar as instruções conforme necessário para o seu projeto específico.

## Tecnologias Utilizadas

Este projeto utiliza várias tecnologias modernas para fornecer uma experiência de desenvolvimento robusta e uma interface de usuário eficiente. Aqui está uma breve descrição de cada uma:

- **Firebase**: Uma plataforma de desenvolvimento de aplicativos móveis e web que oferece uma variedade de serviços, incluindo autenticação de usuários, banco de dados em tempo real e hospedagem de arquivos estáticos. É utilizada neste projeto para gerenciar o banco de dados.

- **React**: Uma biblioteca JavaScript de código aberto para construir interfaces de usuário, especialmente para single-page applications (SPA). React permite a criação de componentes reutilizáveis que gerenciam seu próprio estado, facilitando o desenvolvimento de interfaces interativas.

- **Bootstrap**: Uma biblioteca de código aberto que contém um conjunto de ferramentas para desenvolvimento web front-end, incluindo estilos de design responsivo e componentes de interface de usuário. É utilizada neste projeto para garantir a responsividade e consistência visual em diferentes dispositivos.

- **Ant Design (antd)**: Uma biblioteca de componentes UI de código aberto para React que oferece uma ampla gama de componentes prontos para uso, seguindo as práticas de design da Ant Financial. Os componentes do Ant Design são utilizados para construir uma interface de usuário moderna e elegante neste projeto.

- **React Toasty**: Uma biblioteca React para exibir notificações de forma elegante em um aplicativo web. É utilizada para fornecer feedback visual ao usuário em diferentes eventos e interações dentro do aplicativo.
